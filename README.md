# 🏆 Grandmaster Chess | Premium Experience

A high-end, modern chess game built with vanilla **HTML**, **CSS**, and **JavaScript**. This project focuses on visual excellence, smooth performance, and a premium "Grandmaster" aesthetic.

![Chess Preview](https://via.placeholder.com/800x450/0f172a/38bdf8?text=Grandmaster+Chess+Experience)

## ✨ Features

- **Premium Dark Mode**: A sophisticated navy and slate theme with glassmorphism elements.
- **Smooth Animations**: CSS transitions for piece selection and movement (Phase 2).
- **Crisp SVG Graphics**: High-quality vector pieces that look sharp on any display.
- **Responsive Design**: Playable on desktop, tablet, and mobile browsers.
- **Semantic HTML**: Built with accessibility and SEO best practices in mind.

## 🚀 Getting Started

Simply open `index.html` in any modern web browser to start the experience.

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/SimarSabharwal/simple-chess-game-.git
   ```
2. Navigate to the directory:
   ```bash
   cd simple-chess-game-
   ```
3. Open `index.html` in your browser.

## 🛠️ Built With
- **HTML5**: Semantic structure.
- **CSS3**: Modern layout (Grid/Flexbox) and glassmorphism.
- **JavaScript (ES6+)**: Game logic and state management.
- **Google Fonts**: 'Outfit' and 'Playfair Display'.
